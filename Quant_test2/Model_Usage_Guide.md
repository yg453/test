# 📈 A股量化预测系统 - 模型藏经阁与预测引擎说明

本系统集成了**模型基因打榜库 (Model Manager)** 与 **机构级增量微调预测引擎**。系统会自动为您保存胜率最高、收益最佳的 5 个神经网络训练切片。

## 一、 控制台配置方法

请打开 `debug_stock.py` 文件底部，修改「架构师控制台」：
```python
    RUN_MODE = 'explore' 
    TARGET_CODE = 'sh.600000'
    TARGET_RANK = 1

1. 探索打榜模式 (RUN_MODE = 'explore')
作用：犹如“抽盲盒”。系统随机生成一个种子，从头训练 LSTM，并在 25-26 年测试集回测。如果收益高于库中最后一名，自动覆盖并封存入库。

操作建议：多次运行该模式，让 AI 自己去“优胜劣汰”。

2. 再训练强化模式 (RUN_MODE = 'retrain')
作用：提取神级模型（配合 TARGET_RANK=1），在原基础上继续暴力训练 30 个 Epoch，试图打破原有的收益记录。
针对长时间后更新数据库设计，易至过拟合，须控制时间窗口或采用增量学习策略或调整学习率。

3. 明日实盘微调预测 (RUN_MODE = 'predict')
核心革命：这是实盘赚钱的终极入口。系统会提取 TARGET_RANK 的老模型，拦截最近半年的实盘数据，将学习率调低至 1e-5，快速微调 5 个 Epoch。

优势：完美解决了“老模型不懂新市场”的痛点，兼具十年大局观与三个月短期盘感。

不会保留此微调结果。每次运行后，仍为原始 TARGET_RANK 模型。

输出：直接吐出明天的买卖看多置信度及操作建议。
